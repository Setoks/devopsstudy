Права доступа:
r (read) = 4 - чтение
w (write) = 2 - запись
x (execute) = 1 - выполнение
chmod r <<имя файла>> изменение прав числовой способ
chmod изменение прав символьный способ
Типы файлов 
d - директория
- обычный файл
l - символическая ссылка
c - спец файл символьного устройства
b - спец файл блочного устройства
useradd - создание пользователя
usermod - изменение пользователя
userdel - удаление пользователя
groupadd - создание группы
passwd - смена пароля
